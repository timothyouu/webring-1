# CSUF ACM Webring

A webring for CSUF ACM friends. Share your personal sites, portfolios, and projects!

## Join the webring

1) Add your website to the [index.html](index.html) file:

```html
<li data-lang="en" id="your-name">
	<a href="https://yoursite.com">Your Name</a>
</li>
```

2) Submit a Pull Request!

3) (Optional) Add the webring badge to your site:

```html
<a href="https://webring.lyyeric.tech/#your-name" target="_blank" rel="noopener">
  <img src="https://webring.lyyeric.tech/icon.black.svg" alt="CSUF ACM Webring"/>
</a>
```

Use `icon.white.svg` if your site has a dark background.

### Optional: Add RSS/Banner

```html
<li data-lang="en" id="your-name">
	<a href="https://yoursite.com">Your Name</a>
	<a href="https://yoursite.com/feed.xml" class="rss">rss</a>
	<img loading="lazy" src="https://yoursite.com/button.gif" alt="Your Name" width="88" height="31">
</li>
```
